Creating RasterFrames

From ProjectedExtent

The simplest mechanism for getting a RasterFrame is to use the toRF(tileCols, tileRows) extension method on ProjectedRaster. 

scala> val scene = SinglebandGeoTiff("src/test/resources/L8-B8-Robinson-IL.tiff")
scene: geotrellis.raster.io.geotiff.SinglebandGeoTiff = SinglebandGeoTiff(geotrellis.raster.UShortConstantNoDataArrayTile@5fa6e9e6,Extent(431902.5, 4313647.5, 443512.5, 4321147.5),geotrellis.proj4.CRS$$anon$3@87f96f9f,Tags(Map(AREA_OR_POINT -> POINT),List(Map())),GeoTiffOptions(geotrellis.raster.io.geotiff.Striped@bab52b4,geotrellis.raster.io.geotiff.compression.DeflateCompression$@5168ac09,1,None))

scala> val rf = scene.projectedRaster.toRF(128, 128)
rf: astraea.spark.rasterframes.RasterFrame = [spatial_key: struct<col: int, row: int>, tile: st_tile]

scala> rf.show(5, false)
+-----------+--------------------------------------------------------+

From TileLayerRDD

Another option is to use a GeoTrellis LayerReader, to get a TileLayerRDD for which there’s also a toRF extension method. 

import geotrellis.spark._
val tiledLayer: TileLayerRDD[SpatialKey] = ???
val rf = tiledLayer.toRF
