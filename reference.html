Reference

API Scaladoc can be found here.

Here’s a preliminary list of RasterFrame UDFs available. For the most up to date list, look at the source code for ColumnFunctions. These UDFs are also registered with the SQL engine under the same name but with a st_ prefix (e.g. localMin becomes st_localMin).

Note

Please submit an issue if there’s a particular function you think should be included.

  • explodeTiles - Create a row for each pixel in tile.
  • explodeTileSample - Create a row for each pixel in tile with random sampling.
  • tileDimensions - Query the number of (cols, rows) in a tile.
  • aggHistogram - Compute the full column aggregate floating point histogram.
  • aggStats - Compute the full column aggregate floating point statistics.
  • tileHistogramDouble - Compute tileHistogram of floating point tile values.
  • tileStatsDouble - Compute statistics of tile values.
  • tileMeanDouble - Compute the tile-wise mean
  • tileMean - Compute the tile-wise mean
  • tileHistogram - Compute tileHistogram of tile values.
  • tileStats - Compute statistics of tile values.
  • nodataCells - Counts the number of NoData cells per tile.
  • dataCells - Counts the number of non-NoData cells per tile.
  • localAggStats - Compute cell-local aggregate descriptive statistics for a column of tiles.
  • localAggMax - Compute the cellwise/local max operation between tiles in a column.
  • localAggMin - Compute the cellwise/local min operation between tiles in a column.
  • localAggMean - Compute the cellwise/local mean operation between tiles in a column.
  • localAggCount - Compute the cellwise/local count of non-NoData cells for all tiles in a column.
  • localAdd - Cellwise addition between two tiles.
  • localSubtract - Cellwise subtraction between two tiles.
  • localAlgebra - Perform an arbitrary GeoTrellis LocalTileBinaryOp between two tile columns.
  • renderAscii - Render tile as ASCII string for debugging purposes.
